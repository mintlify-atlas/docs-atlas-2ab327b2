---
title: "Customize Lyrica's text appearance"
description: "Configure how Lyrica displays lyrics on your panel: font size, text color, character limit, translated lyrics mode, and reconnect text behavior."
---

The appearance settings control how lyrics look on your KDE Plasma panel. Open them by right-clicking the Lyrica widget and selecting **Configure Lyrica...**, then go to the **Appearance** tab.

<Tip>
  Use the system theme font size and text color whenever possible. This keeps Lyrica visually consistent with other panel widgets and adapts automatically when you switch KDE themes.
</Tip>

## Settings

<AccordionGroup>
  <Accordion title="Translated lyrics mode" defaultOpen={false}>
    **Config key:** `tlyricMode` ┬Ę **Type:** Int ┬Ę **Default:** `0`

    Controls how Lyrica displays translated lyrics when a translation track (`tlyric`) is available alongside the original.

    | Value | Behavior |
    |-------|----------|
    | `0` | Original lyrics only (default) |
    | `1` | Translation if available, otherwise original |
    | `2` | `Original \| Translation` |
    | `3` | `Translation \| Original` |

    Modes `2` and `3` show both lines at once, separated by a pipe character. Use mode `1` if you prefer reading translations but still want a fallback when no translation exists.
  </Accordion>

  <Accordion title="Character limit" defaultOpen={false}>
    **Config key:** `characterLimit` ┬Ę **Type:** Int ┬Ę **Default:** `60`

    The maximum number of characters to display per lyric line. Lines longer than this limit are truncated with `...`. Adjust this to match the width of your panel or the available space next to other widgets.
  </Accordion>

  <Accordion title="Font size" defaultOpen={false}>
    **Config keys:** `shouldUseDefaultThemeFontSize` (Bool, default `true`), `configuredFontSize` (Int, default `18`)

    By default, Lyrica uses your KDE Plasma theme's font size. Uncheck **Use system font size** to set a custom pixel size instead. The `configuredFontSize` value only takes effect when the system default is disabled.
  </Accordion>

  <Accordion title="Layout height" defaultOpen={false}>
    **Config key:** `layoutHeight` ┬Ę **Type:** Int ┬Ę **Default:** `28`

    The height of the lyrics widget area in pixels. Set this to match the height of your panel so the text fits without being clipped or leaving excess empty space.
  </Accordion>

  <Accordion title="Text color" defaultOpen={false}>
    **Config keys:** `shouldUseDefaultThemeTextColor` (Bool, default `true`), `configuredTextColor` (Color, default `#ffffff`)

    By default, Lyrica inherits your KDE Plasma theme's text color. Uncheck **Use system text color** to specify a custom hex color. The `configuredTextColor` value only takes effect when the system default is disabled.
  </Accordion>

  <Accordion title="Show reconnecting text" defaultOpen={false}>
    **Config key:** `showReconnectingText` ┬Ę **Type:** Bool ┬Ę **Default:** `true`

    When the plasmoid loses its connection to the backend daemon, it displays `[Reconnecting...]` in the panel while it attempts to reconnect. Disable this if you prefer the widget to stay blank during reconnection instead.
  </Accordion>
</AccordionGroup>
